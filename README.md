# scanner2030ad
Scanner 2030 - Andorra

## Requirements

NodeJS > 10.0

## Setup

```
git clone https://github.com/politicalwatch/scanner2030ad.git
cd scanner2030ad
npm install
cp src/config/index.js.example src/config/index.js
```

Finally, edit *config/index.js* file with your specific values.


## Run

```
npm run serve
```
